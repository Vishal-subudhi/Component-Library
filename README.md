# Component Library 🧩
 
A reusable React component library showcasing 10 production-ready UI components built with Tailwind CSS.
 
## Components Built
| Component | Key Concepts |
|-----------|-------------|
| Button | Props, variant mapping, size mapping, disabled state |
| InputField | Controlled inputs, useState, error handling |
| Card | Props, conditional rendering, isPopular badge |
| Alert | Color mapping, type variants, onClose handler |
| Avatar | Conditional rendering, image vs initials fallback |
| Badge | Variant styling, pill shape |
| Modal | children prop, isOpen state, overlay |
| Navbar | Props, Avatar integration |
| Toggle | useState, smooth CSS transitions |
| Tooltip | useState, onMouseEnter/onMouseLeave, absolute positioning |
 
## Tech Stack
- React + Vite
- Tailwind CSS v3
## How to run
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
## Key Patterns Learned
- **Variant mapping:** Using objects instead of if/else for cleaner conditional styles
- **children prop:** Passing JSX between component tags (used in Modal)
- **Controlled inputs:** value + onChange + useState working together
- **Compound components:** Parent managing state, child displaying it
